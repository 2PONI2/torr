ESCAPE THE BACKROOMS

★★★★★ 4.9 / 5.0
Дата выхода: 15 мая 2024
Жанр: Хоррор, Кооператив
Разработчик: Fancy Games
Платформа: Windows
Язык: Русский, Английский

Системные требования (Минимальные)

Операционная система Windows 10 (64-bit)
Процессор Intel Core i5 / AMD FX
Оперативная память 8 ГБ
Видеокарта NVIDIA GeForce GTX 1060
Место на диске 15 ГБ
В этом исследовательском хорроре на 1-4 игроков вам предстоит пройти более 30 жутких уровней закулисья, отстаиваясь от ужасающих существ и пытаясь найти путь домой. Помните: здесь нигде нет безопасности и выжить смогут не все.

Установка:

  • 1. Установите игру
  • Скачать
  • 2. Установите онлайн фикс
  • Скачать

Запуск Мультиплеера:

  • 1. Открываем Steam и входим в свой аккаунт
  • 2. Запускаем Backrooms.exe

Игра по сети:

Создать сервер:

  • 3. Multiplayer → Create Server → Указываем количество игроков → Создаем сервер и ждем их готовности, после чего начинаем игру

Подключиться к серверу:

  • 4. Multiplayer → Присоединяемся к серверу из списка. (Если сервер не виден - присоединяйтесь через приглашение)
