## 📖 О проекте

Этот репозиторий содержит исходный код статического веб-сайта. Проект представляет собой набор HTML-страниц с собственными стилями (`css`) и скриптами (`js`), объединенных в единую структуру.

<details>
<summary>**На данный момент в проекте реализованы следующие страницы:**</summary>

*   **Главная страница:** `index.html` — основная страница сайта.
*   **Администрирование:** `admin.html` — страница для управления.
*   **Донаты:** `donat.html` — страница поддержки.
*   **Страница игры:** `game.html`.
*   **Плейсхолдер страницы игры:** `game_placeholder.html`.
*   **Инструкция:** `instruction.html` — руководство для пользователей.
*   **Конфиденциальность:** `konfident.html` — политика конфиденциальности.
*   **Библиотека:** `lib.html` — библиотека(пока не трогаем).
*   **Правообладатель:** `owner.html` — владелец просит удалить игру. 
*   **Регистрация и вход:** `reg.html` и `sing_in.html`.
*   **Магазин:** `store.html`.
*   **Отказ от ответственности:** `disclaimer_of_liability.html`.
</details>

---

### 📥 Шаг 1. Скачай проект

#### Вариант 1: Через Git (если установлен)

Открой терминал (Командную строку, PowerShell или Terminal) и выполни:

```bash
# Перейти на рабочий стол
cd Desktop

# Склонировать репозиторий
git clone https://github.com/твой-ник/название-репозитория.git

# Перейти в папку проекта
cd название-репозитория
```

---

#### Вариант 2: Скачать ZIP-архив (проще)

1. Перейди по ссылке репозитория.
2. Нажми зелёную кнопку **`<> Code`**.
3. Выбери **Download ZIP**.
4. Распакуй архив на рабочий стол.

---

### 💻 Шаг 2. Открой проект в VS Code

1. Запусти **Visual Studio Code**.
2. Нажми **File → Open Folder...**
3. Выбери папку с проектом.
4. Нажми **Open**.

💡 Либо просто перетащи папку проекта в окно VS Code.

---

### ▶️ Шаг 3. Запусти проект

#### Самый простой способ — Live Server

##### Установка расширения

1. Нажми **Ctrl + Shift + X**
2. В поиске введи **Live Server**
3. Нажми **Install**

### 🔧 Шаг 4. Чекни работоспособность

Теперь можно:

- ✏️ Редактировать файлы HTML, CSS и JavaScript
- 💾 Сохранять изменения (`Ctrl + S`)
- 👀 Сразу видеть результат в браузере

Live Server должен автоматически обновлять страницу после сохранения, обязательно включи автосохранение.

---

### 📤 Шаг 5. Отправка изменений на GitHub

#### Настройка Git (выполняется один раз)

Открой терминал в VS Code (`Ctrl + \``) и выполни:

```bash
git config --global user.name "Твое Имя"

git config --global user.email "твоя-почта@example.com"
```

---

#### Сохранение изменений в GitHub

```bash
# Проверить изменения
git status

# Добавить все файлы
git add .

# Создать коммит
git commit -m "Описание изменений"

# Отправить изменения
git push
```

##### Пример коммита

```bash
git commit -m "Обновил стили главной страницы"
```

🔑 Если Git запросит авторизацию — используй логин GitHub и Personal Access Token (PAT), а не пароль.

---

### 📥 Шаг 6. Получение обновлений

```bash
git pull
```

---

### ⚡ Функции

| Команда | Что делает |
|----------|------------|
| `git status` | Показывает изменённые файлы |
| `git add .` | Добавляет все изменения |
| `git commit -m "текст"` | Создаёт коммит |
| `git push` | Отправляет изменения на GitHub |
| `git pull` | Получает последние изменения |
| `git log` | Показывает историю коммитов |

---

### 🚨 Фиксы

| Проблема | Решение |
|-----------|----------|
| Не запускается Live Server | Переустанови расширение или перезапусти VS Code |
| Не работает `git push` | Проверь авторизацию GitHub и настройки Git |
| Конфликт при `git pull` | Ищи в инете я хз |
| Не видно изменений в браузере | Нажми `Ctrl + F5` для полной перезагрузки |
